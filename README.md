# PHP Delegate Sample


Blog post 
https://codechord.com/2019/03/php-delegate/


Demo run 
```
$ php App/Delegate/Sample.php
$ php App/Delegate/Sample2.php
$ php App/Delegate/Sample3.php
```

or

```
$main = new Sample();
$main->run();

$main2 = new Sample2();
$main2->run();

$main3 = new Sample3();
$main3->run();
```
